# Angel Custom Homes

Luxury custom homebuilder landing page for angelcustomhomes.com.

## Stack
- Next.js
- Tailwind CSS
- Framer Motion
- Cloudflare Pages ready

## Local development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Cloudflare Pages settings
- Framework preset: Next.js
- Build command: `npm run build`
- Build output directory: `.next`
